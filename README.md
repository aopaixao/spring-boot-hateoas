# Descrição - API Rest com Spring HATEOAS
API Rest simples para demonstração da biblioteca Spring HATEOAS

# Funcionalidades Implementadas

* Recursos HATEOAS para os métodos CRUD
* Endpoints CRUD
* Documentação com Springdoc

# Comandos

## Subir a aplicação

```bash
mvn spring-boot:run
```
